# 🇵🇪 Pulso Económico Perú

App web que convierte datos económicos del Banco Central de Reserva del Perú (BCRP) en un análisis personalizado para el ciudadano peruano — usando inteligencia artificial.

## ¿Qué problema resuelve?

El peruano promedio no sabe cómo la inflación, el tipo de cambio o las tasas de interés afectan su bolsillo específico. Los datos del BCRP existen pero son técnicos y difíciles de interpretar.

**Pulso Económico Perú** toma esos datos y los convierte en recomendaciones concretas según tu perfil financiero personal.

## ¿Cómo funciona?

1. Descarga datos en tiempo real del BCRP (tipo de cambio, inflación, tasa interbancaria)
2. El usuario responde 4 preguntas sobre su situación financiera
3. OpenAI GPT-4o-mini genera un análisis personalizado con impactos y acciones concretas

## Stack tecnológico

- **Python** — lenguaje principal
- **Streamlit** — interfaz web
- **Pandas** — procesamiento de datos
- **Plotly** — visualización de tendencias
- **OpenAI API** — generación del análisis personalizado
- **BCRP API** — fuente de datos económicos oficiales
